# Responsives Admin-Dashboard (HTML & CSS)

[🇬🇧 Switch to English Version](README.md)

Ein modernes, klares und vollständig responsives Admin-Dashboard, das von Grund auf mit semantischem HTML5 
und modernem CSS3 (einschließlich CSS Flexbox und Grid-Layouts) entwickelt wurde.
Dieses Projekt zeigt meine Frontend-Entwicklungsfähigkeiten, mein strukturelles Denken und mein Auge für UI/UX-Design.
## 🚀 Funktionen
* **Modernes Dark-Mode-Design:** Eine technologieorientierte, kontrastreiche dunkle Benutzeroberfläche.
* **Sidebar-Navigation:** Eine saubere Navigationsleiste, optimiert für Desktop-Bildschirme.
* **Analytische Karten:** Wichtige Leistungskennzahlen (KPIs) mit visuellen Trend-Indikatoren (Badges).
* **Tabelle für letzte Aktivitäten:** Saubere, strukturierte Datendarstellung durch semantische HTML-Tabellen und Status-Badges.
* **Moderne Layout-Techniken:** Nutzt CSS Flexbox für Ausrichtungen und CSS Grid für ein flüssiges Kartensystem.

## 🛠️ Tech-Stack & gezeigte Fähigkeiten
* **Sprachen:** HTML5, CSS3
* **Tools:** PyCharm / VS Code, Git/GitHub
* **Konzepte:** * Prinzipien des Responsive Designs
  * CSS-Layouting (Flexbox & Grid)
  * Semantisches HTML für bessere Barrierefreiheit (SEO & Screenreader)
  * UI/UX Best Practices (Abstände, Typografie, Komponenten-Styling)
