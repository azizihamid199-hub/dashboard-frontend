# Responsive Admin Dashboard (HTML & CSS)

A modern, clean, and fully responsive Admin Dashboard interface built from scratch using semantic HTML5 and modern CSS3 (including CSS Flexbox and Grid layouts). This project showcases my frontend development skills, structural thinking, and eye for UI/UX design.

## 🚀 Features
* **Modern Dark Mode Design:** A tech-focused, high-contrast dark user interface.
* **Sidebar Navigation:** A clean navigation bar designed for desktop viewports.
* **Analytical Cards:** Key performance indicators (KPIs) with custom visual indicators (trend badges).
* **Recent Activities Table:** Clean, structured data representation using semantic HTML tables and custom status badges.
* **Modern Layout Tech:** Utilizes CSS Flexbox for alignment and CSS Grid for a fluid card system.

## 🛠️ Tech Stack & Skills Demonstrated
* **Languages:** HTML5, CSS3
* **Tools:** PyCharm / VS Code, Git/GitHub
* **Concepts:** * Responsive Design principles
  * CSS Layouting (Flexbox & Grid)
  * Semantic HTML for better accessibility (SEO & Screenreaders)
  * UI/UX best practices (spacing, typography, component styling)
